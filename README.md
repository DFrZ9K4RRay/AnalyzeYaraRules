# AnalyzeYaraRules
## 概述
   Analyze Yara Rules 是用于日常yara规则记录和汇总的一个开源库。我们主要在做的是资源整合以及日常工作收集情报汇总整合。
### 为什么要维护这样一个库？
   补充应急场景下现场扫描需求，不同平台兼容以及部署自动化等原因，我们选择与国际团队思路靠拢，并注意维护团队yara规则共享，同时不断通过积累实现项目维护及推动，赋能一线实施人员，利于现场检查扫描项目快速检测。
